## 💻 About

This is my **personal freelance business website**, designed to showcase my work as a web developer.  

- Based in **Edinburgh, Scotland**  
- Specializes in **responsive, interactive, and user-friendly websites**  
- Demonstrates **projects inside device frames**: users can scroll websites **within the laptop or phone mockup**, giving a realistic preview of projects.

## ✨ Features

- Fully **responsive design** for desktop, tablet, and mobile  
- Interactive **project previews inside device frames**  
- Contact form integration for clients to reach out  
- Clean, maintainable code structure for HTML, CSS, JS, and PHP  
- Modern CSS animations for smooth user experience
